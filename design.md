# Personal Website Design System

## Design Philosophy

### Visual Language
Modern, professional, and technical aesthetic that reflects expertise in data science and engineering. The design balances sophistication with approachability, using clean lines, strategic use of data visualizations, and subtle animations to create an engaging portfolio experience.

### Color Palette
- **Primary**: Deep Navy (#1a1f36) - Professional, trustworthy
- **Secondary**: Warm Copper (#c17817) - Innovation, energy
- **Accent**: Soft Sage (#7c9885) - Growth, balance
- **Background**: Off-white (#fafafa) - Clean, readable
- **Text**: Charcoal (#2d3748) - High contrast, readable
- **Light Gray**: #e2e8f0 - Subtle borders and dividers

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Code Font**: "JetBrains Mono" - Technical monospace for code snippets

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth page transitions and element animations
- **ECharts.js**: Interactive skills radar chart and data visualizations
- **Typed.js**: Typewriter effect for hero section
- **Splitting.js**: Text animation effects
- **p5.js**: Particle background system
- **Pixi.js**: Advanced visual effects for hero section
- **Splide.js**: Project showcase carousel

### Effects & Animations
- **Hero Background**: Subtle particle system with floating geometric shapes
- **Text Effects**: Typewriter animation for main heading, color cycling for emphasis
- **Scroll Animations**: Smooth reveal animations with stagger effects
- **Hover Effects**: 3D tilt on project cards, glow effects on buttons
- **Data Visualization**: Animated radar chart showing skills progression

### Styling Approach
- **Layout**: Grid-based with consistent spacing using 8px base unit
- **Cards**: Subtle shadows with hover elevation effects
- **Buttons**: Rounded corners with gradient backgrounds and hover states
- **Images**: Rounded corners with subtle overlay effects
- **Background**: Consistent off-white with geometric accent patterns

### Header Effect
- Fixed navigation with blur backdrop effect
- Smooth color transition on scroll
- Active state indicators with copper accent
- Mobile-responsive hamburger menu with smooth transitions

## Component Design

### Project Cards
- Clean white background with subtle shadow
- Hover effect: slight elevation and scale transform
- Tech stack tags with colored backgrounds
- Preview images with aspect ratio consistency

### Skills Visualization
- Interactive radar chart with smooth animations
- Hover states revealing detailed information
- Color-coded categories matching design system
- Responsive design for mobile viewing

### Timeline Component
- Horizontal layout with connecting lines
- Interactive nodes with expansion animations
- Professional color coding for different experience types
- Smooth scrolling and focus management

### Contact Form
- Modern input styling with floating labels
- Real-time validation with visual feedback
- Success states with celebratory micro-animations
- Professional button styling with hover effects
